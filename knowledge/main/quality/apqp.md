# APQP — Advanced Product Quality Planning

## Phases

### Phase 1: Planning
- Define scope
- Customer requirements review
- Feasibility assessment
- Project plan
- Team formation (cross-functional)

### Phase 2: Product Design and Development
- Design FMEA
- Design reviews
- Prototype build and test
- Material specifications
- Dimensional specifications

### Phase 3: Process Design and Development
- Process Flow Diagram
- Process FMEA
- Control Plan (prototype, pre-launch, production)
- Work Instructions
- Packaging specifications

### Phase 4: Product and Process Validation
- Production trial run (significant run)
- Measurement Systems Evaluation (MSA)
- Initial Process Capability Study (Ppk ≥ 1.67)
- Production Validation Testing
- PPAP submission

### Phase 5: Feedback, Assessment, Corrective Action
- Continuous improvement
- Reduced variation
- Customer satisfaction monitoring
- Delivery and service support

## Key Deliverables
- Design Record
- Engineering Change Documents
- Design FMEA
- Process Flow Diagram
- Process FMEA
- Control Plan
- MSA Results
- Dimensional Results
- Material/Performance Test Results
- Initial Process Capability Study
- Lab Qualifications
- Appearance Approval Report (AAR)
- Sample Product
- Master Sample
- Checking Aids
- Customer-Specific Requirements
- Part Submission Warrant (PSW)
