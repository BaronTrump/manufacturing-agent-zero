# OEE — Overall Equipment Effectiveness

## Definition
OEE measures manufacturing equipment effectiveness as the product of Availability, Performance, and Quality.

## Formula
```
OEE = Availability × Performance × Quality

Availability = Run Time / Planned Production Time
Performance = (Ideal Cycle Time × Total Parts) / Run Time
Quality = Good Parts / Total Parts
```

## Six Big Losses

### Availability Losses
1. **Equipment Failure (breakdowns)** — unplanned stops
2. **Setup and Adjustment** — changeovers, tooling changes

### Performance Losses
3. **Idling and Minor Stops** — jams, sensors blocked
4. **Reduced Speed** — running below ideal cycle time

### Quality Losses
5. **Process Defects** — scrap, rework during production
6. **Reduced Yield** — startup losses, warm-up scrap

## World-Class Benchmarks
- OEE: 85%+
- Availability: 90%+
- Performance: 95%+
- Quality: 99.9%+

## Calculation Methods

### Method 1: Simple OEE
- Direct calculation from the three factors above

### Method 2: OEE with Availability Only
- Used when performance and quality data are unavailable
- Partial OEE benchmark: 60-65%

### Method 3: Total Effective Equipment Performance (TEEP)
- TEEP = OEE × Loading
- Loading = Planned Production Time / Calendar Time

## Practical Application
- Track OEE by shift, day, week, month
- Use Pareto analysis on Six Big Losses
- Target the largest loss category first
- Implement SMED for changeover reduction
- Implement TPM for breakdown reduction
