# Manufacturing Agent Zero — Skills Index

Skills are reusable procedures the agent can load on demand.

## Available Skills

### `process-capability-study`
Procedure for conducting a process capability study per AIAG SPC manual.
Steps: data collection plan → normality test → control chart → Cp/Cpk calculation → interpretation.

### `ppap-submission-review`
Checklist-based review of PPAP submission packages (19 elements per AIAG PPAP manual).
Returns a gap analysis with required corrections.

### `8d-report-generation`
Guided 8D problem-solving report generator.
D1: Team → D2: Problem Description → D3: Interim Containment → D4: Root Cause → D5: Permanent Corrective Action → D6: Implementation → D7: Preventive → D8: Closure.

### `fmea-workshop`
Facilitated FMEA session guide.
Walks through structure analysis → function analysis → failure analysis → risk analysis → optimization.

### `lean-waste-walk`
Gemba walk checklist for identifying the 8 wastes (DOWNTIME):
Defects, Overproduction, Waiting, Non-utilized talent, Transportation, Inventory, Motion, Excess processing.
