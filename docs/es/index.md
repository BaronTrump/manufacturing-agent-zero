# Manufacturing Agent Zero

Un asistente de IA localizado para la fabricación de piezas de automoción.

## Inicio Rápido
```bash
curl -fsSL https://bash.manufacturing-agent.ai | bash
```

## Características
- Perfiles de agente específicos para fabricación
- Multilingüe: inglés, español, coreano
- Entrene con sus propios datos de empresa
- Se ejecuta completamente en las instalaciones
- Recuperación de conocimiento basada en RAG
