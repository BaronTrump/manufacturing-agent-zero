# Manufacturing Agent Zero

자동차 부품 제조를 위한 지역화된 AI 어시스턴트입니다.

## 빠른 시작
```bash
curl -fsSL https://bash.manufacturing-agent.ai | bash
```

## 기능
- 제조 특화 에이전트 프로필
- 다국어 지원: 영어, 스페인어, 한국어
- 자체 회사 데이터로 학습 가능
- 완전 온프레미스 실행
- RAG 기반 지식 검색
