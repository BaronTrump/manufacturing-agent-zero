# Manufacturing Agent Zero

A localized AI assistant for automotive parts manufacturing.

## Quick Start
```bash
curl -fsSL https://bash.manufacturing-agent.ai | bash
```

## Features
- Manufacturing-specific agent profiles
- Multilingual: English, Spanish, Korean
- Train with your own company data
- Runs fully on-premises
- RAG-based knowledge retrieval
